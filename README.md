# E-commerce API

A REST API for an e-commerce platform, built as a backend portfolio project.

## Project goals

- Build a secure API using TypeScript and Node.js.
- Model a realistic e-commerce workflow: catalog, cart and orders.
- Apply authentication, authorization, validation, tests, documentation and containerization.

## Planned stack

- Node.js + TypeScript + Express
- PostgreSQL + Prisma ORM
- JWT + bcrypt
- Zod
- Swagger / OpenAPI
- Docker and Docker Compose
- Vitest and GitHub Actions

## Main features

- User registration and login
- Roles: `CUSTOMER` and `ADMIN`
- Product and category management
- Stock validation
- Shopping cart management
- Order checkout with a simulated payment flow
- API documentation and automated tests
